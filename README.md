# Personal Portfolio 

Welcome to the repository of my personal portfolio! This is a frontend web application built to showcase my skills as a Fullstack Developer, my featured projects, and a little bit about who I am.

## Tech Stack

This project was intentionally built as a lightweight, fast, and static frontend application using:

*   **Framework:** React / Next.js
*   **Styling:** Tailwind CSS
*   **Data Management:** Local JSON (`data.json`) for dynamic rendering of projects and skills without needing a backend.
*   **Deployment:** Vercel / Netlify

## Features

*   **Responsive Design:** Fully optimized for mobile, tablet, and desktop.
*   **Project Gallery:** Dedicated cards showcasing my Fullstack projects (React, Node.js, Python, Django) with links to live demos and source code.
*   **About Me Section:** A glimpse into my technical background, core skills, and personal hobbies.
*   **Clean Architecture:** Component-driven structure focused on reusability and clean code practices.

## How to Run Locally

If you want to clone this repository and run it locally, follow these steps:

1. Clone the repository:
   
   git clone [repo-name]

2. Navigate to the project directory:

   cd [your-repo-name]

3. Install the dependencies:

   npm install

4. Start the development server:

  npm run dev

5. Open http://localhost:3000 in your browser to see the result.
